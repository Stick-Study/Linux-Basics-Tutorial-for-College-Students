## Linux 的**目录结构**

大多数初学者初见 Linux 的问题一定是“我的C盘去哪了”，“软件安装到哪里去了”以及“我的文件都放在哪里了”，这主要是因为两者不同的目录结构导致的。

这是 Windows 的目录结构：
```
C:
├── Windows                  //系统文件
├── Program Files            //64 位软件安装位置
├── Program Flies (x86)      //32 位软件安装位置
├── Users                    //用户文件夹
```
与此同时，你还会有 D盘，E盘等等很多盘符。<br>
但是 Linux 的目录结构是这样的：
```
/
├── bin   //系统最基本的二进制可执行命令，如 `ls`, `cp`, `mkdir` 等。
├── boot  //系统启动时需要的文件
├── dev   //包含设备文件。在 Linux 中“一切皆文件”，硬盘、鼠标、键盘等硬件都在这里以文件形式存在。
├── etc   //存放系统的**配置文件**
├── lib   //存放系统和 /bin、/sbin 中二进制文件运行所需的共享库文件
├── lib64 //存放系统和 /bin、/sbin 中二进制文件运行所需的共享库文件

├── home  //普通用户的家目录。每个用户都有一个以用户名命名的子目录（如 /home/ubuntu），用于存放个人文档和设置
├── root  //系统管理员（超级用户 root）的家目录，独立于 /home
├── usr   //系统的核心部分之一，包含用户使用的应用程序和文件
├── opt   //存放第三方厂商提供的软件包

├── sbin  //存放系统管理员使用的重要管理命令，如 `fdisk`, `reboot`, `iptables` 等
├── var   //存放内容经常发生变化的文件
├── tmp   //存放程序运行时产生的临时文件。**通常系统重启时会清理此目录**
├── mnt   //手动挂载的文件系统挂载点
├── media //自动挂载的可移动设备（如 U 盘、光驱）

├── proc  //包含系统进程信息的虚拟文件系统
├── sys   //提供内核、设备和驱动程序的层级信息
└── run   //存放系统启动以来描述系统运行信息的临时文件
```
~~当然我写这么多仅仅是为了满足你的好奇心~~

你会看到 Linux 的特殊之处在于他没有各种**盘符**，所有东西（包括硬件设备、各种外置储存设备）都在`/`目录下，也就是**根目录**下。

你只需要记住你的个人文件都在 `/home/用户名` 下，其他都交给系统来完成就好了。

## 终端

在 Windows 中，大多数操作是通过图形界面完成的，例如点击文件、拖动窗口等。

但在 Linux 中，**终端（Terminal）** 是主要的操作方式。

终端本质上是：

> 一个用于输入命令、并让系统执行这些命令的窗口。

简单来说，就是使用命令来和系统交互。

## Shell

Shell 是运行在终端中的**命令解释器**。当我们输入命令后，Shell 就会接受这个命令，然后调用系统程序来执行这个命令。

常见的 Shell 有：
- bash（Ubuntu 默认）
- zsh (建议搭配 [Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh) 使用)

在终端输入：
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
便可自动安装 Oh My Zsh

在本教程中，默认使用 bash。~~因为期末考试会用这个~~

## 命令

刚才说过，我们主要使用各种命令来和系统交互。其中，命令的基本格式如下所示：
```
命令 [选项] [参数]
```
例如：
```
ls -l /home //以长格式列出 /home 路径下的文件 
```

**注意：Linux 是严格区分大小写的，大小写字母一般有着不同的意义！**

接下来我将介绍一些常用的命令：

### 查看当前所在目录

可以使用 **P**rint **W**orking **D**irectory 命令

```
pwd
```
终端会输出：
```
/home/ubuntu
```
代表着此时你在 `/home/ubuntu` 目录下。

### 查看当前用户

Linux 是一个多用户系统，你可以登陆不同的用户，包括 root 用户。可以使用如下命令来查看当前用户：
```
whoami
```
终端会输出：
```
ubuntu
```
也可能会是
```
root
```
第一个代表着此时你正在使用 ubuntu 用户。而第二个代表着你正在使用 root 用户。

**注意：root 用户是系统中权限最高的用户，不正确的操作可能会导致系统灰飞烟灭哦！**

### 查找手册

当我们不知道一个命令如何使用的时候，可以使用以下命令来查看手册：

```
命令 --help
```
或
```
man 命令
```
按 `q` 退出手册。

### 自动补全

按 `Tab` 键可以自动补全命令或路径。

### 历史命令

- ↑：上一条命令
- ↓：下一条命令

### 清屏

```
clear
```
或使用快捷键：
```
Ctrl + L
```
